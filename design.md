# FO.CU Platform — Design System

## Brand
- **Name:** FO.CU by The Body Lab
- **Tagline:** A tua plataforma de transformação
- **Audience:** Mulheres, clientes do desafio FO.CU, membros pagos

## Colors
```
--cream:      #FFF5EE   (background principal)
--orange:     #F07A30   (primary, CTAs, highlights)
--orange-lt:  #F4A261   (hover states, gradients suaves)
--peach:      #FFE5D0   (cards, badges, sub-backgrounds)
--black:      #1A1A1A   (texto principal)
--gray:       #6B6B6B   (texto secundário)
--gray-lt:    #F5F0EC   (borders, dividers)
--white:      #FFFFFF   (surfaces, cards)
--green:      #4A7C59   (leaf accent, tags nutrição)
```

## Typography
- **Font:** Poppins (Google Fonts)
- **Display:** Poppins 800 (headlines, hero)
- **Headings:** Poppins 700
- **Sub-headings:** Poppins 600
- **Body:** Poppins 400
- **Small/Labels:** Poppins 500 uppercase tracked

## Layout
- Max width: 1280px
- Sidebar navigation (desktop): 260px fixed left
- Main content: remaining width with padding
- Mobile: bottom tab bar + full-width content

## Component Style
- Cards: white bg, border 1px solid #F5F0EC, radius 16px, shadow subtle
- Buttons: radius 12px, height 44px, bold 600
- Primary button: orange bg, white text
- Ghost button: transparent, orange border + text
- Tags/Pills: peach bg, orange text, radius 24px, text-xs uppercase
- Input fields: cream bg, orange focus ring
- Navigation active: orange text + left accent bar (desktop)

## Navigation Sections
1. **Início** (dashboard/home)
2. **Vídeos** (YouTube embeds por categorias)
3. **Conteúdos** (PDFs & Ebooks download)
4. **Nutrição**
   - Contador de Calorias
   - Rastreador de Alimentos
   - Receitas Saudáveis

## Aesthetic
- Feminine, warm, motivacional
- Clean — muito espaço em branco (cream)
- Orange como energia + urgência
- Ícones line-style
- Imagens: peach/warm tones
- No-fluff — conteúdo direto
